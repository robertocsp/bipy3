Quem é o Marviin



Marviin é um robô para facebook messenger integrado a um sistema de controle de pedidos voltado para bares, restaurantes, pizzarias, lanchonetes e etc. Agora clientes podem realizar seus pedidos digitando mensagens de texto, interagindo com o robô Marviin, que se encarrega de enviar para o seu estabelecimento o pedido. Não precisa instalar nenhum app e nem acessar nenhum site. Tudo pelo seu facebook messenger. Além disso, possibilita uma base de clientes para que seja possível um relacionamento cliente x estabelecimento.
